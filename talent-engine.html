How it works

Three pillars.
Built for the Dutch market.

01 / SOURCE
Multi-channel discovery
AI monitors Tweakers, ICTerGezocht, OfferZen, LinkedIn, GitHub, Magnet.me, Nationale Vacaturebank and Intermediair — surfacing qualified candidates daily. No scraping. Only legitimate API & RSS sources or inbound applications.
02 / MATCH
AI scoring + written rationale
Each candidate scored against SF-ICT's real job requirements with a written AI note explaining why — skills, experience, Dutch-language fit, location, availability signals. You see the reasoning, not a black box.
03 / ENGAGE
Human-approved outreach
AI drafts hyper-personalised messages referencing each candidate's actual background. Every message is reviewed and approved by you before it sends. Zero spam. GDPR-legal. Scheduling handled through your existing calendar.

A phased approach.

The Talent Engine ships in three phases. Each phase delivers value on its own — and lays the technical foundation for SAGE, so nothing is wasted.

Phase 1 · MVP
Foundation & First Sources
Weeks 1–6
  • Integrate LinkedIn, Tweakers, ICTerGezocht RSS / API sources
  • Operator console with candidate pipeline + AI match scoring
  • AI outreach drafting with human-approval workflow
  • Azure EU deployment + GDPR compliance baseline
  • Integration with existing SF-ICT calendar & email
Phase 2 · Expansion
Full Dutch Coverage + ATS
Weeks 7–12
  • Add GitHub, OfferZen, Magnet.me, Nationale Vacaturebank
  • Optional: Recruitee integration as full ATS
  • Candidate CRM — call lists, notes, scheduled follow-ups
  • Automated coding assessments via HackerRank / TestGorilla
  • Performance analytics: time-to-hire, source ROI
Phase 3 · Convergence
Bridge to SAGE
Month 4+
  • Shared AI foundation with SAGE (same Azure OpenAI, same vector store)
  • Internal resource allocation: match engineers to tickets by skill
  • Skills-gap analysis: where should SF-ICT hire next?
  • Client account intelligence fed from recruitment data
  • Unified operator dashboard: recruitment + operations + SAGE
